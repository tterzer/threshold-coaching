Precise CdA builder
Pick specific rides, then pick the laps that are the real comparable loop (e.g. exclude the ride to/from the park). Each ride gets one CdA fit pooled across its selected laps.
If a ride only shows 1-2 giant laps instead of real loops, intervals.icu auto-detected power-based intervals instead of using the device's lap-button presses. Fix on intervals.icu: open that ride → Actions → Use Laps → Keep All Laps → reopen the picker here.
Race predictor
Upload a course GPX file, then adjust target power to see predicted finish time update. Integrates speed continuously across the course (momentum carries from descents into climbs, etc.) -- assumes a single constant power target, not a terrain-optimized variable pacing strategy, and doesn't account for wind.

Aero (CdA / Crr) estimate
Estimates CdA/Crr from steady rides using the same virtual-elevation method as Best Bike Split (no wind data, calm-air assumption). Needs rides with real hills — pancake-flat rides can't separate drag from rolling resistance. Mixing road and TT bike rides will make this bounce around a lot since they have very different drag — use the name filter above if ride titles distinguish them. Click "Build trend" to analyze.
